
# React + Vite
## [Live here](https://appointment-swart.vercel.app/)

## Key Features:
 - Fetch data from api 
 - Also search and get data of that name.
 - use tailwind css for styling

## Prerequisites

Node version 18.7.x



## Install packages

```shell
npm i
```

## Start the dev server

```shell
npm run dev
```

## Build the page

```shell
npm run build
```
